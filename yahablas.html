Ya Hablas — Language Academy

A full-featured language learning app with 2,000+ questions per language, multiple levels, and 10+ exercise types

Role Designer & Developer
Year 2025
Status Live

About this Project

A full-featured language learning app I designed and built from scratch. Each language course contains over 2,000 questions spread across multiple proficiency levels (A1–C1), organized into thematic units. The platform includes 10 distinct exercise types — Multiple Choice, True or False, Vocabulary Drill, Fill in the Blank, Matching, Opposites, Sentence Ordering, Minimal Pairs, Error Correction, and Dialogue — plus 5 practice modes: Speed Round, Listening, Speaking, Dictation, and Shadowing.

Currently live with English and Spanish courses, featuring XP progression, streak tracking, a competitive leaderboard, daily challenges, a personal word bank, and spaced repetition. The interface adapts to each language with a warm, encouraging UX designed to keep learners engaged long-term.

Tech Stack

HTML / CSS / JS Webflow Web Speech API Spaced Repetition Multi-language Responsive Design
