Ya Hablas — Language Academy

A complete language learning ecosystem with Duolingo-style gamification

Role Designer & Developer
Year 2025
Status Live
yahablasidiomas.com

About this Project

A complete language learning ecosystem I designed and built from scratch. The platform features Duolingo-style gamification with XP progression, streak tracking, achievement badges, a competitive leaderboard, spaced repetition algorithms, daily challenges, and a personal word bank.

It supports multiple languages across different proficiency levels — from beginner to advanced. The interface adapts between Spanish and English, with a warm, encouraging UX designed to keep learners engaged.

Tech Stack

HTML / CSS / JS Webflow Gamification API Spaced Repetition Multi-language Responsive Design

Key Features

  • XP, levels, badges & streaks
  • Leaderboard & competition
  • Spaced repetition algorithm
  • Daily challenges
  • Personal word bank
  • Multi-language support
