About this Project

A centralized management system I built for FOM Hochschule to handle hundreds of weekly lecture recordings. The dashboard features three distinct views — main overview, recording detail (CMP), and room/studio detail (STM) — with advanced filtering by date, time, type, studio, and attendance status.

It integrates with Panopto and Moodle, showing real-time recording status, documentation tracking, and feedback loop management. The system replaced manual spreadsheet workflows and dramatically reduced errors.
