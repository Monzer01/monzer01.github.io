Smart Feedback System

An intelligent feedback tool with voice-to-text and automatic data pre-filling

Role Designer & Developer
Year 2024
Status Internal

About this Project

An intelligent feedback and reporting system designed for the studio operations team at FOM. Studio supervisors receive a unique URL with pre-filled event data (ID, date, time, room, course, lecturer) — no manual entry needed.

The form captures structured feedback on technical issues, organizational deviations, lecturer notes, and editing requests. A standout feature: voice-to-text recording with automatic transcription, so supervisors can dictate feedback hands-free. All submissions feed directly into the ZDF VMS dashboard.

Tech Stack

HTML / CSS / JS Web Speech API URL Parameters VMS Integration Responsive Design

Key Features

  • Voice-to-text dictation
  • Auto-filled event data from URL
  • Structured issue categories
  • Direct VMS dashboard feed
  • Pre-locked context fields
  • Mobile-optimized for studio use
